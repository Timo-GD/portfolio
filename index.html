Hold the Line

Hold the Line is a space invaders clone made as an assignment for the Amsterdam University of Applied Sciences.
It was my first project for the Amsterdam University of Applied Sciences and the project was done solo.

Body Block

Body Block is an original platformer made as an assignment for the Amsterdam University of Applied Sciences.
It was my second project for the Amsterdam University of Applied Sciences and the project was done in a group of two.

Mirror Mirror on the Board

Mirror Mirror on the Board is an original multiplayer board game made as an assignment for the Amsterdam University of Applied Sciences.
It was my fourth project for the Amsterdam University of Applied Sciences and the project was done in a group of four.

Flappy Birb
