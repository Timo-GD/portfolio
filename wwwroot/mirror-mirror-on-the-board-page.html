Mirror Mirror on the Board

Mirror Mirror on the board is a multiplayer board game made using the mongame framework. It was made within two months together with Thijmen Waitz, Tim Houtzager and Kendahl Turk.
Mirror Mirror on the board is a game in which you have to use mirrors, t-splits and prisms to redirect the laser to hit the other player whilst also making sure the other player doesnâ€™t hit you.

It was the second project that I worked on that used the monogame framework. It also was the first project that I worked on that used multiplayer.
Incorporating multiplayer was a difficult task. But we eventually used a client server model. Using datapackets that we send to the server which then sends the correct response to both users. Sadly due to the game running on the servers of the University the game is no longer playable.

The other difficult part of making the game was the grid system and making all the objects you could place here respond correctly to the inputs. We worked on this problem using a tile and grid system.
The grid system checks the position of the tile you want to place, and whether it is possible. And then the tile which has a specific tag based on which tile it is will do the logic of how it has to interact with the surroundings.
In this system the player, the laser and the obstacles are all tiles but with different tags.
